## Основные проекты

<blockquote>

### [botix](https://github.com/KiraFlux/botix)

Открытый образовательный робот с 3D-печатной конструкцией и полным набором исходников

</blockquote>

<blockquote>

### [ESP32-DJC](https://github.com/KiraFlux/ESP32-DJC)

DIY девайс на ESP32 с расчётом на функциональность наземной MAVLink станции.

</blockquote>

<blockquote>

### [flix](https://github.com/KiraFlux/flix)

Форк [`okalachev/flix`](https://github.com/okalachev/flix) образовательного мини-квадрокоптера на ESP32. [`dev/espnow`](https://github.com/KiraFlux/flix/tree/dev/espnow)

Любительская версия: [Klyax](https://github.com/KiraFlux/Klyax) - работа над CAD моделью дрона

</blockquote>

<blockquote>

### **[KiraFlux-ToolBox](https://github.com/KiraFlux/KiraFlux-Toolkit)**
C++20 фреймворк для ESP32, построенный на статическом полиморфизме (CRTP) и миксинах. Включает графический стек, реактивно-событийную UI подсистему, драйверы некоторых дисплеев, сенсоров, моторов, а также математические инструменты. 
Разрабатывается для личных проектов, покрывается тестами, примерами, но пока не претендует на production-готовность.

</blockquote>

<blockquote>

### [MizLang](https://github.com/KiraFlux/MizLang)

Экспериментальный язык, упор на comptime вычисления с типами. Пока заморожен.  

</blockquote>
